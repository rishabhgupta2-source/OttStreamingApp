# OttStreamingApp

Monorepo / workspace for **StreamList**, a React Native TMDB discovery app.

## Mobile app

All app code, scripts, and documentation for running and developing StreamList live here:

**[`Task2/StreamList/README.md`](Task2/StreamList/README.md)**

Quick path:

```bash
cd Task2/StreamList
npm install
cp .env.example .env   # then add your TMDB token
npm start
```
